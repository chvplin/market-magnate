Market Magnate — purchasable item icons
========================================

Single source of truth: icon-manifest.json (regenerated by tools/mm-icon-build.mjs).

Runtime loader + resolvers: mm-icon-assets.js (loaded from game.html before main game script).

Regenerate manifest + JS after changing item NAMES in game.html:
  node tools/mm-icon-build.mjs

Regenerate / replace PNG files (512x512):
  powershell -ExecutionPolicy Bypass -File tools/generate-item-icon-pngs.ps1

Premium art prompts (replace stub PNGs): see prompts/*.md

Debug missing images in console:
  window.MM_DEBUG_ICONS = true
