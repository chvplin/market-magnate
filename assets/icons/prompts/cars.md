# Image generation — cars (512×512 PNG)

Use for each filename in `assets/icons/cars/*.png` (replace gradient stubs).

**Style:** realistic luxury automotive product photography, three-quarter front view, studio rim light, dark graphite background (#0e0e14–#1a1a24), subtle floor reflection, premium hypercar / sedan / SUV detail, no logos that infringe real brands—evocative original designs.

**Framing:** centered vehicle, ~75% frame width, consistent camera height.

**Negative:** white studio sweep, cartoon, low-poly, watermark, text overlay, collage, steering-wheel-only macro unless item is explicitly “interior”.
